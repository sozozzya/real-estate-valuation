# Real Estate Valuation Agent

В приложении реализована **целевая ridge-регрессия с априорными оценками** для модели:

\[
V_i = \beta S_i + \alpha Q_i + \varepsilon_i
\]

где:
- \(V_i\) — цена объекта,
- \(S_i\) — площадь дома,
- \(Q_i\) — площадь участка,
- \(\beta\) — удельная стоимость дома,
- \(\alpha\) — удельная стоимость участка.

## Единая математическая постановка

Используются априорные оценки из прошлого периода \(\beta_0, \alpha_0\). Оценка параметров ищется как решение:

\[
(\beta^*, \alpha^*) = \arg\min_{\beta,\alpha}
\sum_{i=1}^{N}(V_i - \beta S_i - \alpha Q_i)^2
+ \lambda_\beta (\beta-\beta_0)^2
+ \lambda_\alpha (\alpha-\alpha_0)^2.
\]

В матричном виде:

\[
\theta^* = (X^T X + \Lambda)^{-1}(X^T y + \Lambda \theta_0),
\]

где:
- \(X=[S\;Q]\),
- \(\theta=(\beta,\alpha)^T\),
- \(\theta_0=(\beta_0,\alpha_0)^T\),
- \(\Lambda = \text{diag}(\lambda_\beta, \lambda_\alpha)\).

## Что считает агент

1. Оценки \(\beta^*\), \(\alpha^*\).
2. Метрики качества: RSS, MSE, RMSE, MAE, MAPE, \(R^2\).
3. Стандартные ошибки и 95% доверительные интервалы для \(\beta\), \(\alpha\).
4. Формулу прогноза: `V = beta * S + alpha * Q`.
5. Использованные \(\lambda_\beta\), \(\lambda_\alpha\) (ручной или автоматический выбор).

## Режимы выбора регуляризации

- **Ручной**: пользователь задаёт `lambda_beta` и `lambda_alpha`.
- **Автоматический**: старт от дисперсионных оценок и подбор пары \((\lambda_\beta,\lambda_\alpha)\) по LOOCV на небольшой сетке.
